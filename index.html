Introduction

Zen Kommerce is an open source PHP shopping cart system written in the Kohana framework.

Features

  • Guest Checkout
  • One Page Checkout
  • Order Fulfillment
  • Dynamic Pricing
    • Product Quantity Discounts
    • Catalog Promotions
    • Shopping Cart Promotions
  • Coupon Codes
  • Products
    • Tags
    • Wishlist
    • Ratings/Reviews
  • Custom Tax Rules
  • Reports
    • Sales
    • Tax
  • Shipping Weight
  • Order Fulfillment
  • Downloadable Products
  • Digital Viewing Subscription
  • Templatable
  • PCI Compliant
  • Search Engine Optimization (SEO)
  • Responsive Design
  • Social Sharing
  • Import/Export Products
  • Backup/Restore
